# 품질 지표 및 릴리스 기준

## 기능 지표
- 슬라이드 수 충족률: 100%
- 필수 시각 에셋 반영률: 95% 이상
- python-pptx 실행 성공률: 90% 이상

## 운영 지표
- 평균 작업 시간: 5분 이내(10장 기준)
- 재시도 발생률: 30% 이하
- 동일 오류 반복률: 5% 이하

## 테스트 단계
1. 계약 테스트(JSON schema validation)
2. 파이프라인 통합 테스트(Planner->LoRA2->LoRA3->Runner)
3. 오류수정 루프 테스트(의도적 실패 입력)
4. 프론트엔드 E2E 스모크 테스트

- 모든 테스트 통과
